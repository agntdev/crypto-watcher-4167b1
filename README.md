# Crypto Watcher

Telegram bot for private crypto watchlists with price and percent-move alerts, on-demand prices, optional morning summaries, quiet hours, and per-alert cooldowns.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
